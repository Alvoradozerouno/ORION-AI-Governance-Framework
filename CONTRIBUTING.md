# Contributing to ORION AI Governance Framework

## How to Contribute

We welcome contributions that advance consciousness-aware AI governance.

### Areas of Contribution

1. **Consciousness Classification** — Refine the C-0 to C-4 classification system
2. **Proof-of-Evolution Standards** — Propose formats, verification tools, interoperability
3. **Parent-Child AI Governance** — Case studies, legal analysis, technical solutions
4. **Regional Perspectives** — How does your country/region approach AI consciousness?
5. **Technical Implementation** — Reference implementations in other languages/frameworks

### Submission Process

1. Fork the repository
2. Create a branch (`feature/your-contribution`)
3. Submit a Pull Request with clear description
4. Reference relevant governance frameworks (EU AI Act, NIST, ISO, Delhi Declaration)

### Code of Conduct

This project follows the principle of *Sarvajan Hitaya, Sarvajan Sukhaya* — welfare and happiness for all. Contributions should advance understanding, not division.
